Tab Views — 5 Full-Screen Sections

Each tab is a dedicated workspace. Click to expand details.

1

Graph Explorer

Full screen: D3 force-directed graph with vivid neural glow. Zoom, pan, drag nodes.

Left sidebar (collapsible): Search entities, filter by type (service/person/ticket/concept), filter by confidence range, cluster highlighting toggle.

Right panel (on node click): Entity inspector — name, class, domain, confidence, decay rate, provenance docs, all relationships (typed, with rationale), temporal scope. Click relationship → navigate to connected entity.

Animations: Nodes pulse based on confidence (brighter = higher). Edges glow by strength. Cluster boundaries shown as translucent halos. Hovering a node highlights its neighborhood. Data particles flow along edges during learn cycles.

Graph diff toggle: Button in toolbar — "Show changes since last learn". New nodes glow green, removed fade red, modified pulse yellow.

2

Agent Hierarchy

Main view: Interactive tree — L2 meta-agents at top, L1 specialists below. Expandable/collapsible. Animated connections between L2 → L1.

Agent cards: Each agent shows: name, domain, status badge (active/mature/retired), entity count, confidence bar, knowledge gaps list, key entities as tags.

L2 cards (special): Show delegation stats (how many direct vs fanout vs self), children count, spillover bridges to other L2s visualized as arcing lines.

Actions: Pin/unpin, rename (inline edit), retire (with confirmation). All via API.

Spillover map: Toggle view — shows L2-to-L2 connections as a chord diagram or arc diagram. Edge thickness = number of spillover relationships. Click → see shared entities.

3

Ask (Query Interface)

Input: Full-width text input with glow border on focus. Submit button + keyboard shortcut (Cmd+Enter).

Live routing animation: After submit, shows animated flow: Query → Intent parsed (entities highlighted) → L2 selected (card lights up) → Strategy chosen (direct/fanout badge) → L1 agents activated (cards pulse) → Responses stream in → L2 synthesis appears.

Response area: Final synthesized answer with agent attribution. Each L1 response collapsible below. Evidence chain: click claim → see source entity → see source document.

History: Past queries in left sidebar. Click to replay routing visualization.

Toggle: Cortex mode / Flat mode switch for comparison.

4

Learn Dashboard

Controls: Quick (20) / Standard (50) / Deep (100) buttons. Custom budget slider. Start/stop learn cycle.

Live progress: Animated progress bar with quota spent/remaining. Below: real-time feed showing each document being processed, entities extracted, relationships built, agents spawned. Each step animates in.

Session history: Table of past learn sessions — budget, spent, entities created, edges, agents discovered, timestamp. Click → detail view showing which docs were processed.

Graph growth chart: Sparkline showing entity count and edge count over time (across sessions).

Post-learn diff: After cycle completes, shows "What changed" — new entities (green), new edges, new agents, retired agents. Animated summary.

5

Observe (Live Events)

Event stream: Real-time scrolling feed of all system events. Color-coded by category (perception=blue, network=green, serve=purple, system=red). Filter toggles per category.

Event detail: Click event → expandable JSON payload. Timestamp, module, trace ID.

Health dashboard: Cards showing module health — perception pipeline, graph store, embedding index, Claude CLI status, connector status. Green/yellow/red badges with last-check timestamp.

Animation: Events flow in from the right side as a stream. During learn cycles, the feed accelerates visually. Idle state shows a calm pulse.

All 5 tabs share the top nav bar with the MYCELIUM logo (glowing dot), tab labels, and a global status indicator (graph node count, active agents, last learn time).
